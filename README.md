# Finance & Life Dashboard

Dashboard personal construido con Next.js App Router, React, Tailwind CSS, Lucide React y Prisma listo para Supabase/PostgreSQL.

## Primeros pasos

```bash
npm install
cp .env.example .env
npm run prisma:generate
npm run dev
```

Para conectar Supabase, coloca `DATABASE_URL` y `DIRECT_URL` en `.env`.
